Всем привет.
Это рубрика - обзор классов Java SE версии 11.
В данном видеоролике будет рассмотрен класс Boolean пакета java.lang.

[показать на весь экран второй кадр презентации]
Класс Boolean не нужно явно импортировать, так как он расположен в пакете java.lang, который импортируется автоматически в любой ваш код.

[показать на весь экран третий кадр презентации]
Класс Boolean появился в Java SE версии один точка ноль.
Наследуется он от класса Object и реализует он интерфейсы Serializable и Comparable

[показать на весь экран четвёртый кадр презентации]
Класс Boolean является обёрткой над примитивным типом boolean и содержит внутри себя одно нестатическое поле примитивного типа boolean. Так же класс содержит методы для преобразования примитивного типа boolean в чтроку и обратно.

[показать на весь экран пятый кадр презентации]
